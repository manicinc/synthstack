# Pricing & Features (Community vs Pro)

This page explains what’s included in SynthStack Community and what’s added in SynthStack Pro.

For current pricing and plans, see: https://synthstack.app/pricing

## Feature Overview

| Area | Community | Pro |
|------|-----------|-----|
| Vue + Quasar cross-platform app | ✅ | ✅ |
| Directus CMS integration | ✅ | ✅ |
| Stripe billing foundation | ✅ | ✅ |
| Authentication (Supabase / Local) | ✅ | ✅ |
| Deployment docs (VPS/AWS/GCP) | ✅ | ✅ |
| AI Copilot / agents / orchestration | ⚠️ Limited or disabled by default | ✅ |
| Workflows / advanced automation | ❌ | ✅ |
| Referrals / credits / feature flags | ❌ | ✅ |
