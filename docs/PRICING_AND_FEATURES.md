## Pricing Tiers

SynthStack offers four pricing tiers to suit different needs:

| Tier | Price | Best For |
|------|-------|----------|
| **Free** | $0/month | Trying out SynthStack, hobby projects |
| **Starter** | $19/month | Individual developers, small projects |
| **Premium** | $49/month | Professional developers, production apps |
| **Lifetime** | $499 one-time | Long-term users, power users |

### Free Tier

Perfect for getting started with SynthStack:

- 5,000 free credits to start
- Basic AI Copilot access
- Up to 3 agents
- Community support
- Core workflow features
- Rate limits: 10 requests/min
- 50,000 tokens/day max

### Starter Tier ($19/month)

For individual developers who need more capacity:

- 50,000 credits/month included
- All AI models (GPT-4, Claude 3.5, etc.)
- Up to 10 agents
- Priority email support
- Advanced workflow features
- Custom integrations
- Rate limits: 50 requests/min
- 500,000 tokens/day max

### Premium Tier ($49/month)

For professionals building production applications:

- 200,000 credits/month included
- **BYOK (Bring Your Own Keys)** - Use your own API keys!
- All AI models and features
- Unlimited agents
- Priority support with SLA
- Advanced analytics
- Team collaboration features
- Custom branding (white label)
- Rate limits: 100 requests/min
- Unlimited tokens (with BYOK)

### Lifetime Tier ($499 one-time)

One-time payment for lifetime access:

- All Premium features forever
- 150,000 credits/month included for life
- BYOK (Bring Your Own Keys)
- Priority support for life
- Early access to new features
- Lifetime software updates
- Transferable license
- One-time payment (no recurring fees!)

## BYOK (Bring Your Own Keys)

**Available for Premium & Lifetime tiers only**

### What is BYOK?

BYOK allows you to use your own API keys from OpenAI, Anthropic, and other providers instead of SynthStack's internal keys.

### Benefits

- Save Money - Pay provider's direct pricing
- Unlimited Usage - No SynthStack token limits
- Bypass Rate Limits - Provider limits only
- Custom Pricing - Use negotiated rates
- Full Control - Manage your own quota

See [FAQ_BYOK.md](./FAQ_BYOK.md) for complete documentation.
